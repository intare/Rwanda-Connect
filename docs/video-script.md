## Technical Notes for Video Production:

### Screen Recording Tips:
1. **Device Setup**: Use a clean iPhone/Android simulator or physical device
2. **Screen Resolution**: Record in 1080p minimum for clarity
3. **Interaction Speed**: Move deliberately but naturally through the app
4. **Timing**: Each section should align with voice-over duration
5. **Highlights**: Use subtle animations to draw attention to key features

### Key Features to Emphasize:
- **Verification badges** on opportunities and mentors
- **Real-time data** with member counts and statistics
- **Professional design** with clean, modern interface
- **Comprehensive filtering** and search capabilities
- **Community engagement** features (likes, bookmarks, shares)
- **Cross-platform compatibility**

### Voice-Over Style:
- **Professional but warm** tone
- **Clear pronunciation** of Rwandan names and places
- **Moderate pace** to allow visual comprehension
- **Enthusiastic emphasis** on key benefits and features

### Video Length: 
Approximately 5:30 minutes total, perfect for social media and presentation use.
