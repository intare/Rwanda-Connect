# Release Plan

## Milestones
- M1: UX and architecture baseline.
- M2: Core flows implemented (auth, news, opportunities, events, community).
- M3: Beta release and feedback.
- M4: Public launch.

## Beta Scope
- Internal test group.
- Manual content updates.
- Core analytics tracking.

## App Store Checklist
- Privacy policy and terms links.
- App screenshots and descriptions.
- Crash reporting and analytics enabled.

## Launch Criteria
- No P1 bugs open.
- Core flows pass smoke tests.
- Performance meets target on mid-tier devices.
