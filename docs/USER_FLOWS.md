# User Flows

## 1. Register and Onboard
1. Launch app.
2. Tap Register.
3. Enter name, email, password.
4. Confirm account creation.
5. Onboarding: select interests, set location.
6. Arrive at Home feed.

## 2. Login and Resume Session
1. Launch app.
2. If session exists, go to Home.
3. If not, enter credentials and sign in.
4. Arrive at Home feed.

## 3. Browse Opportunities and Apply
1. Open Opportunities tab.
2. Use filters or search.
3. Open a listing detail.
4. Tap Apply to open external link.
5. If gated, show upgrade prompt.

## 4. Bookmark Opportunity (Phase 2)
1. Open opportunity detail.
2. Tap Bookmark.
3. Confirm in Saved list.

## 5. RSVP to Event
1. Open Events tab.
2. Search or filter by location and type.
3. Open event detail.
4. Tap RSVP intent.
5. Confirm RSVP recorded.

## 6. Create Community Post
1. Open Community tab.
2. Tap New Post.
3. Enter text and submit.
4. Post appears in list.

## 7. Edit Profile
1. Open Profile tab.
2. Update fields and interests.
3. Save and return to previous screen.

## 8. Subscription Upgrade
1. Trigger gated action.
2. View plan options.
3. Select trial or paid plan.
4. Confirm plan state saved.
