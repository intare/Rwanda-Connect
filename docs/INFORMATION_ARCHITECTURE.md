# Information Architecture

## How to Use This Doc
- Use this to align UX navigation, screens, and primary flows.
- Validate app tabs and routes match this structure before implementation.
- Cross-check feature scope with `PRD.md` and build modules in `architecture.md`.

## Navigation Model
- Bottom tabs (MVP):
  - Home
  - Opportunities
  - Events
  - Community
  - Profile

## Home
- News list
- Featured opportunities
- Quick actions

## Opportunities
- List with filters and search
- Detail view
- Apply link

## Events
- List with search and filters
- Event detail
- RSVP intent

## Community
- Discussion list
- Create post

## Profile
- Basic profile details
- Interests
- Subscription status

## Phase 2 Additions
- Properties (tab or deep link from Home)
- Playbook (tab or section in Home)
