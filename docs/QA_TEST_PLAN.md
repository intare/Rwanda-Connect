# QA Test Plan

## Devices
- Android: low, mid, high tier phones.
- iOS: small, medium, large screens.

## Smoke Tests
- Register and login.
- Onboarding completes or skips.
- Home loads content.
- Opportunities list and detail open.
- Events list and RSVP intent works.
- Community post creation works.
- Profile edit persists.

## Regression Areas
- Auth and session persistence.
- Network error handling.
- Navigation across tabs.
- Form validation.

## Accessibility
- Large text settings.
- Screen reader labels on primary actions.
- Touch targets 44x44.
