# Entitlements

## Plans
- Free
- Trial (30 days)
- Monthly
- Yearly

## Feature Matrix
| Feature | Free | Trial | Monthly | Yearly |
| --- | --- | --- | --- | --- |
| Browse news | Yes | Yes | Yes | Yes |
| View opportunities list | Yes | Yes | Yes | Yes |
| Apply to opportunities | No | Yes | Yes | Yes |
| Save/bookmark items | No | Yes | Yes | Yes |
| Event RSVP | Yes | Yes | Yes | Yes |
| Community posting | Yes | Yes | Yes | Yes |
| Mentorship access | No | Yes | Yes | Yes |
| Premium analytics | No | No | Yes | Yes |
| Priority support | No | No | Yes | Yes |

## Limits
| Limit | Free | Trial | Monthly | Yearly |
| --- | --- | --- | --- | --- |
| Opportunity applications | 0 | Unlimited | Unlimited | Unlimited |
| Bookmarks | 0 | 100 | Unlimited | Unlimited |
| Mentorship requests | 0 | 3 | Unlimited | Unlimited |

## Gating Rules
- If user is Free, block "Apply to opportunities" and show paywall.
- If trial expired, downgrade to Free and enforce Free limits.
- If payment fails, enter grace period and show billing prompt.

## Paywall Triggers
- Primary: Apply to opportunities.
- Secondary (optional): Bookmark items or mentorship access.
